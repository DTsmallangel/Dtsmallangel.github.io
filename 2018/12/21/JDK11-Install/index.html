JDK11 下载安装及环境配置教程

1. 搜索JDK进入Oracle官网 (https://www.oracle.com/technetwork/java/javase/downloads/index.html) 下载jdk11.点击下载按钮下载jdk


下载windows版本的.exe文件

2. 点击下载的JDK安装

先在想要安装JDK的地方创建一个文件夹命名为JDK，点击下载好的安装包进行安装，路径选择刚才我们创建的那个文件夹。

3. 配置环境变量

安装好之后，打开环境变量配置界面，修改两个地方

  • JAVA_HOME:在系统变量中创建JAVA_HOME路径为JDK路径
  • Path：在Path中添加JAVA_HOME\bin

4. 检验是否配置成功

打开dos窗口输入java -version和javac -version，如果可以返回相应的版本号则说明配置成功
